# Changelog

All notable changes to the Splitrail VS Code extension will be documented in this file.

## [0.1.0] - 2025-11-30

### Added

- Initial release
- Activity Bar dashboard with usage statistics
- Status bar showing today's token usage and costs
- Live updates when AI tool data changes
- Support for multiple AI coding tools:
  - Claude Code
  - Codex CLI
  - Gemini CLI
  - GitHub Copilot
  - Cline, Roo Code, Kilo Code
  - Qwen Code
  - OpenCode
- Time range filtering (Today, Week, Month, All Time)
- Cloud upload integration with Splitrail Cloud
- Commands for refresh, upload, and opening cloud dashboard
- Configurable CLI path and live updates settings
